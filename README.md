# Run in develop

## With npm
```bash 
  git clone git@github.com:rafasfz/battleship.git

  cd battleship

  # install dependeces
  npm install

  # run in develop
  npm run dev
```

## With yarn
```bash
  git clone git@github.com:rafasfz/battleship.git

  cd battleship

  # install dependeces
  yarn

  # run in develop
  yarn dev
```

You can test the application in production <a href="https://battleship-rafasfz.herokuapp.com/">here</a>
